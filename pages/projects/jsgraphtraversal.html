JavaScript Graph Traversal.

JavaScript

This project uses a prebuilt dataset to simulate graph traversal in the context of airport departure flight paths, it does this using depth and breadth first searching and logs the consequent big O complexity.

This JavaScript file uses depth-first and breadth-first searching algorithms to find routes from one airport in a given list to another. It prints this route to console and logs the Big O complexity associated with finding the result using that searching algorithm. The destination airport can be changed at line 73 and 103 where the functions are called.
