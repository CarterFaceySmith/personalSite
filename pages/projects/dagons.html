Dagon's Mountain./p>

Java

Dagon's Mountain is a lightweight, fully Java-based roleplaying game that runs locally in the user's terminal.

The player will choose from one of three unique classes and embark on a quest in this rogue-like RPG in order to conquer the titular Dagon's Mountain. This project was created in the first weeks of me learning Java and was made to practice fundamental skills, insturctions for running the game can be found in the GitHub repository linked above.
