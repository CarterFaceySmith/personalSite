Euclidean GCF Solver.

Python

A lightweight application that uses Euclid's algorithm to compute the greatest common factor of any two numbers.

This was the first Python project I made and was primarily practice for syntax new to me, the program runs locally in the user's terminal and includes error/exception handling. The instructions for running the program can be found in the GitHub repository linked above.
